API documentation
There is only one kind of API request supported. You can make this request to get a book's title, author, year, average rating on this website and the review count by providing the isbn of the book in a GET request.
The format of your request should be as follows: 
             https://book-review-haris314.herokuapp.com/api/<isbn>
The response to the request will be as folows: 
            {
                "author": "Jennifer Niven", 
                "average_score": 2.83, 
                "isbn": "0385755929", 
                "review_count": 6, 
                "title": "Holding Up the Universe", 
                "year": 2016
            }
if the isbn given is 0385755929, for instance.
For a wrong isbn, or an isbn which does not exist in the database of the website, the response will be as follows together with the error code 404. 
            {
                "isbn": "_INVALID"
            }
